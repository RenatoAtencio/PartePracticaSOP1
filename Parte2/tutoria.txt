para pasar las variables en el .env a variables de entorno debes poner en consola
    set -o allexport
    source Archivo.env
    export
el export te muestra todas las variables de entorno, si ves la lista deberias de ver
las variables que definiste en el .env

ahora en tu codigo deberias de poder usar el getenv, lo debes usar asi:
    getenv("NombreVariable")
    por ejemplo getenv("PATH_FILES_IN") te deberia devolver Parte2/Input
    en el caso de AMOUNT_THREADS no se si te lo devuelve como int o como un string, ahi
        lo conviertes si es necesario

estas variables de entorno no se mantiene si reinicias el pc, o la consola o cierras el visual
por ende debes hacer los pasos del comienzo siempre

recueda que debes entar en la carpeta de Parte2 

en el punto .env cambie Prueba1_Parte2/Input a Parte2/Input y 
    Prueba1_Parte2/Output a Parte2/Output
